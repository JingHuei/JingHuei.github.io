菜鳥的前端之路 — 又往前邁開一步了

發表於2021-12-10|更新於2021-12-15
|閱讀量:

一直以來網頁設計我只會編輯 HTML ，CSS 是什麼我不太清楚，至於什麼是 RWD 網站，一開始我根本就不知道，但了解之後，我期許自己有天也能獨立做出一個 RWD 網站。

每到接近年末，就會有很多人開始審視今年做了什麼、去年立的目標有沒有實現，今年回顧起這一年自己的成長，覺得特別的有成就感。

其實小時候就對網頁設計蠻有興趣，但一直沒有特別去學和深入研究，在學期間學的網頁設計程度只會編輯 HTML 的部分，曾經學過 CSS 但後來完全忘了怎麼使用。

去年誤打誤撞、有幸進入大公司，成為維護公司官網的負責人，當初面試時挺懷疑的，這麼大的公司的官網網頁維護職位，真的只需要我這種三角貓功夫的程度嗎？

引用一段話

連自己都懷疑自己的程度能否在該職位適任。

結果我被錄取了，很開心的同時也抱著很擔心的心情報到。

事實上因為大公司沿用初期建置的老舊網站，而且工作內容大部分都是週期性重複的，前手也都有留下工作步驟教學及紙本紀錄，所以這個職位還真的只需要一個會編輯簡單 HTML 靜態網頁的人就能應付大部分的工作了。

以剛出社會得到的第二份工作來說，能順利進大公司做著符合自己能力範圍的輕鬆工作，還能享受很多以前公司沒有的福利，對我來說超級幸運和幸福。

但好景不常，因為疫情的緣故我遇到許多前所未有的難題，很多連前手都沒經歷過的變動，都在我任職期間全發生了，需要由我去完成這些任務，雖然身為一個負責維護官網的人這樣說有點奇怪，但我真的不太會網頁設計，非原本作業範疇的變動，我看著程式碼研究了老半天，還是完全不知道怎麼處理才好…….

後來這些難題都交由另一個對網頁設計比較專業的同事和資訊管理部的同事去解決了。

經過幾次經驗，我清楚自己的程度有多糟，為了補足不懂的網頁設計技能知識，我試著上網查資料、看技術性文章，但還是不足以在工作上應用，好幾度沮喪自己能力不足又不太懂網路文章內容，也試著想上課增進實力，但有嚴重拖延症又極度被動的我，給自己很多不去上課的理由，一拖再拖，就過了一年。

新的一年開始了，又給自己很多的期許，五月疫情大爆發，變得不適合出門，公司也開始居家辦公，那時的我鐵了心要讓自己在這段關在家裡的時間進步，趁學習的衝動還在，直接報名了網頁設計的線上課程，再來就依課程指示按部就班的認真上課交作業。

這些努力慢慢見效了，現在的我在工作時更看得懂網頁程式碼，也懂得怎麼改寫可以更好、和同事溝通時更聽得懂她說的專業名詞及內容、對偶發性的網頁問題更懂得怎麼找出錯誤和如何應對解決、更讀得懂網路上的技術文章，覺得一切努力都非常值得，心裡非常踏實、有成就感！

最近遇到要將原本委託廣告公司製作的網站遷移回公司主機的難題，同事將這個任務交給我，原本以為只要把檔案放到我們公司的主機上就可以了，沒想到這個任務比我想像的還難很多，因為廣告公司提供了網頁的原始檔，但編寫環境和使用的語言跟我們完全不一樣，導致即使有原始檔案卻無法完整還原網頁。

同事給我兩個方向去選擇，改寫廠商的檔案或者自己從無到有重新寫出一樣的網站，並問我比較偏向哪一個，我則回答：「其實我兩個都不會．．．」，不過真要選的話我後來選了自己重新開始，畢竟自己從無到有做出來才能清楚知道檔案裡有什麼，很感謝她讓我有練習的機會，但同時我心裡害怕得一邊發抖，很怕自己做不出來還讓反倒給同事添加更多麻煩．．．

無論如何勇敢的同事將整個任務交給我了，我當然也要盡所有的心力想辦法去完成！

其實這項任務非常為難我，廠商給的原始檔案一打開我完全看不懂，著手一個多禮拜了還是對於怎麼使用 Bootstrap 寫出一個響應式網站感到非常困惑，到最後只好使用最笨的方法　—　打開 Devtools 把所有 HTML 和 CSS 一行一行打下來，最後只完成一個不會動的靜態網頁，剩下的進度還在不斷努力和焦慮，等這個大任務結束，我再來抽空分享這次執行任務遇到的難關和心得吧！

每年年底總在回首自己當年的收穫及成長，在這看似因為疫情被打亂一切生活秩序的一年，很多人抱怨著不能出去玩，但我認為危機就是轉機，正因為疫情無法出門玩，反而讓我能夠定下心好好的在家裡認真進修，今年是我成長最多的一年，也期望自己能繼續保持學習的動力，持續進步！
